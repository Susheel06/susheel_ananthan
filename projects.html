Projects

RASA based NLU Chat bot - April, 2020

  • Built a restaurant chatbot using RASA (NLU framework) where based on user intent, the bot returns top 5 rated restaurant list based on the cuisine and location chosen by the user.
  • Trained our model to extract cuisine, location intent from the user statement. Used Bing map API to extract location id. Further with location id and cuisine, using Zomato API we returned top 5 restaurants.

Smart Irrigation - April, 2020

  • This project incorporates cloud based smart irrigation where the micro-controller actuates desired control action on the water pump based on data transmitted from the field transmitters and weather API.
  • The flow of information between the micro-controller and the cloud is supported by stable network services such as LoRaWAN or Sigfox. Parallelly data from weather API is taken to know possibility of rain to avoid irrigation on that day.
  • The project also incorporates Human Machine Interface(HMI) to monitor data such as moisture content in soil, level/volume of water present in reservoir and other related parameters, to pre-set threshold value of desired parameters to be maintained, also an option of automatic mode and manual mode of operation.

2 DOF Robotic Arm - September, 2017

  • Worked with Robotics club of NIT Trichy to build a 2 DOF Robotic arm. The 2 DOF Robot module is connected to two rotary Servo Base Units which control a 4-bar linkage system.
  • The goal of the 2 DOF Robot experiment is to manipulate the X-Y position of a four-bar linkage end effector and trace the path of the given input co-ordinates accordingly.

Piano Tile Robot - March, 2019

  • The project ‘Piano Tile Robot’ is an approach to play the android game “Piano Tiles” autonomously using Arduino Uno.
  • The circuit is built using LDR, to differentiate between black and white tiles, then processing the input signal from LDR using Arduino controller and actuating the touch on the smart phone using relay circuit by creating a voltage drop on the point of smartphone, if the black tile is detected, using aluminum foil.
